# RESTful API with GoLang on Gin

- In this project, you will build a RESTful API server with two endpoints. Your example project will be a repository of data about vintage jazz records.

## Built With

- Go
- Gin
- Lot of love.

### Prerequisites

- go1.19.3 windows/amd64

### Setup

```
- Clone repository

- go get .
```

### Run server

```
- go run .
```
